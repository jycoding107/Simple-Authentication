Introduction

Asp.Net MVC 5 Simple Authentication is a web application which includes authentication fucntions such as login, sign up, view profile, edit profile, forgot password, change password, and so on. You can view the detailed features here. This source code enables the developers to speed up their development process by simply copying and pasting the code to their project. All the forms in the web application have a responsive design.

Features

  • Login
  • Sign Up
  • Email Confirmation upon Sign Up
  • My Profile
  • Forgot Password
  • Change Password
  • Edit Profile:
    • Upon changing username and username input is validated, the user have to login again with the new username.
    • Upon changing email and the email input is validated, the user have to confirm the new email and re-login.
    • Upon changing profile picture, new image will override the old image.
  • Upload Profile Picture:
    • Validation on Duplication of image file name
    • Validation on file name (cannot contain special characters)
  • Various Sign-Up Validations:
    • Check whether username already exists
    • Check whether email already exists
    • Email format validation
    • Username validation (minimum length 4, spacing and special characters are not allowed)
    • Name validation (only allow letters)
    • Password validation (minimum length 6)
    • Password and confirm password matching validation
    • Required fields validation
  • Terms and Conditions Pop Up
  • Success/Error Message Alert (Alert will automatically disappear after 1.5 seconds)

Code Editing

  • Replace browser's tab icon to your custom icon
    1. Prepare your icon image file in .png format
    2. Rename your custom icon image file to "favicon"
    3. On Solution Explorer, right click on SimpleAuthentication project > Add > Existing Item... > Select your favicon.png file > Replace
  • Replace logo on navigation bar
    1. Prepare your logo image file in .png format
    2. Rename your image file to "logo"
    3. On Solution Explorer, right click on Images folder > Add > Existing Item... > Select your logo.png file > Replace
  • Edit logo's width and height
    1. Go to CustomStyle.css inside Content folder
    2. Search for 'logoTopNavBar' class, you can modify the width and height so that it fit with your logo's dimension. (Tips: you can press Ctrl+F and enter logoTopNavBar to search for it quickly)
  • Edit Email Template
    1. There are total of 3 email template used in this project. Confirm email upon sign up, Update email address confirmation, and Reset password.
    2. Go to Controllers/AccountController.cs
    3. Search for GetConfirmEmailTemplate function if you want to edit email content for 'Confirm email upon sign up'. This is triggered when user sign up an account.
    4. Search for GetConfirmEmailChangingTemplate function if you want to edit email content for 'Update email address confirmation'. This is triggered when user change their email address in 'Edit Profile' page.
    5. Search for GetForgotPasswordEmailTemplate function if you want to edit email content for 'Reset Password'. This is triggered when user submit email address in 'Forgot Password' page.
    6. Each function has the email subject and body which you can edit.
    7. For now, the email template for each function is as shown in the images below.
      • Confirm email upon sign up
      • Confirm email when changing email address in Edit Profile page.
      • Reset Password.
  • Edit button color
    1. Go to Content/w3css.css
    2. Search for 'changed' (I have commented 'changed' for the line which you can change the button's background-color or color)
    3. Edit the background-color, color and so on
  • Edit footer at home page
    1. Go to Views/Home/Index
    2. Search for 'footer' and edit the portion of code as shown in the image below.
